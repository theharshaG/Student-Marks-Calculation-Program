# Student-Marks-Calculation-Program
Simple Student Marks Calculation Program in Python

## Project Description
This is a simple Python program created while learning Python basics.
The program accepts marks for multiple subjects and calculates total,
average, highest mark, and lowest mark.

## How the Program Works
- User enters number of subjects
- User enters marks for each subject
- Program calculates total and average
- Highest and lowest marks are displayed
- Final result is shown as PASS or FAIL

## Concepts Used
- Lists
- For loop
- Conditional statements
- Built-in functions (sum, max, min)
- User input handling

## Learning Outcomes
- Learned how to use lists in Python
- Understood loop-based input
- Practiced basic calculations
- Improved logical thinking

## How to Run
1. Open terminal or command prompt
2. Run the file:python marks_calculation.py
